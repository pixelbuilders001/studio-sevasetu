# **App Name**: SevaSetu

## Core Features:

- Service Category Selection: Allow users to select a service category (e.g., Phone Repair, AC Repair).
- Problem Selection (Dynamic): Dynamically display relevant problems based on the selected service category.
- Price Estimation: Show fixed inspection charges and estimated repair price range.
- Booking Details Form: Collect customer details (Name, Mobile, Address, Time Slot).
- Booking Confirmation: Display Booking ID and confirmation message after booking.
- Technician Assignment Prediction: Predict the most suitable technician based on problem description and technician availability using an AI tool.
- WhatsApp Support Integration: Provide quick access to WhatsApp support via button.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and professionalism.
- Background color: Very light blue (#F0F9FF).
- Accent color: Yellow-orange (#F59E0B) for CTAs and highlights to grab attention.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern yet warm feel.
- Use simple, recognizable icons for each service category.
- Mobile-first, clean layout with minimal steps to reduce booking friction.
- Subtle animations for button hovers and loading states to enhance usability.
